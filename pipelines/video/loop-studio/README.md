# loop-studio

Study notes on Loop Studio (the $97 bespoke video-edit skill set from build-loop.ai), kept to inform
visuals-flow v2. Loop Studio itself is installed outside this repo at `~/.claude-personal/skills/`;
this folder is just the analysis.

## What's here

- **[how-loop-studio-works.md](how-loop-studio-works.md)** — the mechanics, step by step (Steps 0 to
  8): how Loop Studio takes an idea to a finished video, with worked examples and a per-step contrast
  against visuals-flow. Read this to understand the system.
- **[2026-07-23-critique-and-v2-handoff.md](2026-07-23-critique-and-v2-handoff.md)** — the critique
  and cold-start handoff for the v2 brainstorm: what visuals-flow already does better, the ranked
  steal-list (final-video reviewer, sound+mix layer, enacted cards, whole-video throughline), the four
  open questions, the fresh-install fixes, and running state. Read this to decide what to build.

## The short version

Loop Studio made the opposite bet to visuals-flow: bespoke per-beat authoring instead of templates.
Higher craft ceiling, slower and artisanal, not machine-enforced. visuals-flow is the more mature
production system (scale, cost, determinism, linted rules, editor handoff). The three things worth
taking from Loop Studio are the layers visuals-flow lacks: the whole-video concept step, a sound and
mix stage, and a reviewer that runs on the finished video.
