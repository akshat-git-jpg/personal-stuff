# Pronunciation / spelling map

The source of truth for how names and terms should be **spelled** (and **spoken**) in the
final transcript. Step 2 applies every entry here. This file is meant to grow — whenever a
new name shows up in a video, add it, and every future video gets it right for free.

## How to use
1. In step 2, find each name in the transcript and replace the ASR's version with the
   **Correct** spelling.
2. If a correctly-spelled name still mispronounces in the TTS, use the **Spoken-as** column
   (write that phonetic form in the final transcript instead).
3. Name not listed (or unsure)? **Web-search to verify** the real spelling first, fix it, and
   **add a row here.** Only ask the human if the search is inconclusive.

## Numbers / versions — patterns (not a term list)
Apply the pattern, no row needed:
- `1.1` → "one point one", `3.1` → "three point one", `1.2.7` → "one-two-seven"
- `100` → "one hundred", `240` → "two hundred forty", `$56` → "fifty-six dollars"
- `16 by 9` → "sixteen by nine", `50%` → "fifty percent"
