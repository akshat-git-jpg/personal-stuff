### Tutorial Maker

**Input:** Video topic + notes - $\color{Orange}{Manager-Manual}$

**Output:** Final tutorial (3 screen recordings), Instruction for video editor.

**Steps:**

1. Draft script outline from input
2. Review — outline approval - $\color{Orange}{Manager-Manual}$
3. Record tutorial (screen + voiceover)
4. Review — tutorial approval - $\color{Orange}{Manager-Manual}$

**Handoff to:** Video Editor

### Video Editor

**Input:** Instruction file, Tutorial Video, notes, yt meta data - title, description (with link tracker) - $\color{Orange}{Manager-Manual}$

**Output:** Final video uploaded as draft.

**Steps:**

1. From tutorial - gets mp3 file - Makes AI avatar using mp3 file
2. Final video edit with tutorial and avatar
3. Review — complete video - $\color{Orange}{Manager-Manual}$
4. Upload video on yt as draft.

**Handoff to:** Manager
