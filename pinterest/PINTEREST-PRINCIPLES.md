# Pinterest Principles — READ THIS before any Pinterest decision

Canonical rules for every Pinterest skill (make-post, research, board). Niche-agnostic.
Distilled from current (2026) Pinterest-marketing best practice. Keep it short; edit as we learn.

## Core mental model
- Pinterest is a **visual SEARCH engine** (like Google), NOT social media. Never import
  Instagram/TikTok virality tactics — high intent, not entertainment.
- 2026 ranking = **relevancy + quality + engagement** (clicks, saves, session time).
- A pin's only job is to earn an **outbound click** → freebie → email list → sale.
  Impressions/views are vanity. The metric that matters is **outbound clicks**.
  CTR = clicks ÷ impressions; healthy ≈ 0.5–2% (aim 2–3%+). Recipes click higher;
  inspo niches (fashion/nails) lower.
- The winning test for ANY pin: **could a stranger tell what it's about in under 3 seconds?**
- Impressions but no clicks = a **conversion** problem (title/image/keywords), not distribution —
  the algorithm is already working.

## 1. Keywords (the fuel — no keywords = no discovery)
- Find them **from Pinterest itself**, not from Google-SEO instinct (intent/phrasing differ):
  - **Search bar**: type the term, read autocomplete + the guided/colored tiles (ranked by real
    volume). Niche down: `healthy recipes → easy → dinner → veg`.
  - **Pinterest Trends** (trends.pinterest.com): search volume, seasonality, filter by country.
  - **Competitor pins**: open high-ranking **organic** pins (skip ads), read the keywords in their
    title + description. Trick: open a pin's URL while **logged out** — Pinterest exposes its own
    keyword annotations for that pin.
- **Specific beats broad**: "10-minute home workouts" not "fitness"; "high-protein keto breakfast"
  not "recipe". Pinners arrive with high intent.
- Write keywords as **natural sentences**. No comma-stuffed lists.
- It's a **long game (~6 months)**. Start with ~10 targeted keywords, reuse them, review monthly
  (and check whether the keyword or the image drove the win).

## 2. Alignment (the 2026 rule that ties it together)
- **Everything points to ONE topic**: account name, bio, board title, board description, pin title,
  pin description, on-image text, AND the image itself.
- Pinterest ranks **clarity across the whole account**, not a single pin.
  **One clear, aligned pin beats 10 vague ones.** Clarity > volume.
- Reuse the **exact keyword phrase** consistently — don't alternate "budget tracker" / "money tips"
  / "financial planning" for the same thing.

## 3. The image (the single biggest click lever)
- Pinterest's AI **reads the image** (on-pin text, objects, context). Put the **main keyword as
  on-image text**.
- The image must literally **show the result/payoff**: plated meal > stock background;
  visible planner > blank desk.
- Must **stop the scroll in ~1 second**. A beautiful-but-vague image fails — people save it as
  inspiration and never click. A **text overlay** (title or CTA) gives them a reason to click.
- **Readable on a phone**: large, high-contrast text. Test by shrinking the design — if you can't
  read it small, neither can a thumb-scroller.
- Our specs: **1000×1500 (2:3) @2x**; infographics may run taller.

## 4. The copy — 4 click triggers (use at least 2 per pin)
- **Clarity** — instantly who it's for + what it is ("step-by-step keto checklist",
  "keto for complete beginners").
- **Payoff** — a specific, worth-it outcome (what they GET, not "because it's pretty").
- **Curiosity** — the "wait, what?" hook ("you're doing keto wrong", "I tried X so you don't have
  to"). This is the **#1 click driver**.
- **Trust / ease** — looks legit + obvious low-risk next step ("download the free guide",
  show a product mockup).

## 5. Structure (account / boards / pins)
- **Business account** (converts easily from personal; unlocks Trends + analytics).
- **Profile name**: descriptive + keyworded (e.g. "Kushal · Keto Recipes & Meal Plans").
  Can't use the literal word "Pinterest".
- **Bio**: 1–2 sentences, keyword-rich — role + niche + a credibility hook.
- **Boards**: one keyword-optimized board per sub-category; **specific titles** ("Keto Breakfast",
  not "Food Ideas"); keyword-rich description; **2–3 hashtags**. Refresh the year in titles yearly.
- **Pin**: researched keyword **title**; keyword-rich **description** (natural variations + what
  happens after the click) + ~2 hashtags + a **CTA**; correct **destination link**; the **right
  board**; **3–5 tags**; **alt text** describing the visuals (**under 500 chars**).

## Mistakes to avoid (hard rules)
- ❌ Treating Pinterest like social media / chasing views.
- ❌ Keyword stuffing or comma-separated keyword lists.
- ❌ Aesthetic-trend titles over search-intent titles — "Life Reset" ❌ vs
  "Sunday Reset Routine Checklist" ✅.
- ❌ Inconsistent wording across pin / board / profile.
- ❌ Too broad ("fitness", "recipe", "food").
- ❌ A pretty-but-vague image with no text overlay and no shown result.
- ❌ Small or low-contrast on-image text.
- ❌ Treating posting frequency/volume as the strategy.
- ❌ Forgetting the outbound link / the freebie→email funnel.
- ℹ️ Putting your name in pin titles is **optional** — skip it early (often more impressions without
  it); only add it once the brand is established.

## Our funnel (always pull toward this)
**pin → landing page / free guide → email list → PDF sale.** Every pin should move someone one step
along it. The link and the offer are not optional.
