# my-yt — personal YouTube channel notes

Free-form markdown notes about the channel: content system, topic ideas, automation backlog, hiring thoughts. Not consumed by any script. Some files are intentionally empty placeholders.

## Layout

```
my-yt/
├── analysis.md
├── automations-list.md
├── topic-list.md
├── topic-research.md
├── hiring.md
└── ContentCreationSystem/
    ├── InitialFlow.md
    ├── BigComparisonVideos.md
    ├── SmallComparisonVideos.md
    ├── DemoVideos.md
    └── CouponCodeVideo.md
```

## Convention

Treat as a scratchpad. No required structure. Empty files are fine — they reserve a name for future content.
