# tutorial-vo

The tutorial maker's self-serve TTS Worker UI.

## Owner Setup

1. Create D1 database: `npx wrangler d1 create tutorial-vo-db`
2. Update `wrangler.toml` with the generated `database_id`.
3. Apply schema: `npx wrangler d1 migrations apply tutorial-vo-db --remote`
4. Create R2 bucket: `npx wrangler r2 bucket create tutorial-vo-audio`
5. Set secrets: `npx wrangler secret put ADMIN_TOKEN` (and `LINK_SECRET`, `MODAL_TTS_URL`, `MODAL_TTS_TOKEN`, `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`)
6. Deploy: `npm run deploy`

Then update `INFRA.md` and `my-hosted-sites.md`.

## Freelancer Link

Freelancers access the UI via token links generated by the pipeline: `https://vo.agrolloo.com/v/<slug>?t=<token>`
