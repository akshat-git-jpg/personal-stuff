# YouTube production — TODO / parking lot

## Improve the OUTLINE before recording, not just polish after
Status: idea, revisit later.

Step 3 (script polish) always works on a transcript that's already welded to a recorded demo,
so it can only sharpen delivery — it can't fix weak *structure* without desyncing the screen.
The higher-leverage fix is upstream: tighten the outline before the tutorial maker records, or
have Claude expand their bullet points into a tight script they record to.

Tradeoff: the current model is freestyle (no script read), chosen for cost and naturalness.
Moving to a scripted / teleprompter record changes that and may feel less natural. Worth it
only if we want to raise the script-quality ceiling beyond what post-hoc polish can reach.

Revisit when script quality (not delivery) becomes the bottleneck.
