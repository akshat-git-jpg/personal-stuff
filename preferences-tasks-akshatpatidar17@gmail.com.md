# My Scheduling Preferences

## Task Management Rules

- Never merge similar tasks into a single task. Only remove exact duplicates (same task appearing more than once). Each task must stay individual so it can be independently ticked off.
- When organizing tasks into subtask categories, keep every task as its own line item — never combine two tasks into one description.
- Do not create subtask categories or groupings within a list unless the user explicitly asks for it. Present tasks as a flat list by default.
